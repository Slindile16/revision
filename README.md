# revision
📌 Overview

This project contains Python practice exercises focused on core programming concepts.
The goal of the project is to strengthen problem-solving skills by implementing functions and testing them using Python's built-in unittest module.

The exercises simulate common tasks that appear in programming assessments and coding interviews.

🧠 Topics Covered

-Conditional Statements

-Functions

-Basic Loops

-Advanced Loops

-Simple Algorithms

-unit Testing with unittest



🛠 Skills Demonstrated

Writing clean and reusable Python functions

Using conditional logic to make decisions

Iterating through data using loops

Implementing simple algorithms

Writing and running unit tests

Running Python programs using the command line

📂 Project Structure
.
├── revision.py        # Contains the function implementations
├── test_revision.py   # Unit tests for the functions
└── README.md          # Project documentation
▶️ How to Run the Program

Run the Python file from the terminal:

python3 revision.py
🧪 Running the Unit Tests

To run all unit tests:

python3 -m unittest

or

python3 -m unittest test_revision.py
